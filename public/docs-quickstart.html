Интеграция Libermall ID за 5 минут

От нуля до работающего «Войти через Libermall ID» на вашем сайте.

Шаги

1. Получите client_id 2. Установите SDK 3. Настройте redirect 4. Добавьте кнопку 5. Обработайте callback

1. Получите client_id

Напишите на hello@libermall.com с информацией:

  • Название и URL сервиса
  • Redirect URIs (одна или несколько)
  • Нужные scopes: openid profile email минимум

Получите client_id и client_secret в течение суток.

2. Установите SDK

npm install openid-client
# или для Laravel
composer require laravel/socialite
# или для Python
pip install authlib

3. Настройте redirect

В вашем приложении настройте OAuth-клиент с этими значениями:

LIBERMALL_ID_HOST=https://id.libermall.com
LIBERMALL_ID_CLIENT_ID=<полученный_id>
LIBERMALL_ID_CLIENT_SECRET=<полученный_secret>
LIBERMALL_ID_REDIRECT=https://yourapp.com/oauth/libermall/callback

5. Обработайте callback

Пример для Express:

app.get('/oauth/libermall', (req, res) => {
  const url = `${process.env.LIBERMALL_ID_HOST}/login/oauth/authorize?` +
    `response_type=code&client_id=${process.env.LIBERMALL_ID_CLIENT_ID}` +
    `&redirect_uri=${encodeURIComponent(process.env.LIBERMALL_ID_REDIRECT)}` +
    `&state=${crypto.randomUUID()}&scope=openid+profile+email`;
  res.redirect(url);
});

app.get('/oauth/libermall/callback', async (req, res) => {
  const { code } = req.query;
  const tokenRes = await fetch(`${process.env.LIBERMALL_ID_HOST}/api/login/oauth/access_token`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
    body: new URLSearchParams({
      grant_type: 'authorization_code',
      code,
      client_id: process.env.LIBERMALL_ID_CLIENT_ID,
      client_secret: process.env.LIBERMALL_ID_CLIENT_SECRET,
      redirect_uri: process.env.LIBERMALL_ID_REDIRECT,
    }),
  });
  const { access_token } = await tokenRes.json();
  const userRes = await fetch(`${process.env.LIBERMALL_ID_HOST}/api/userinfo`, {
    headers: { Authorization: `Bearer ${access_token}` },
  });
  const user = await userRes.json();
  req.session.user = user;
  res.redirect('/');
});
