Use cases

Для каких продуктов подходит Libermall ID

5 сценариев применения: от dApp до классического SaaS.

Web3 dApps

TON Connect login с поддержкой Tonkeeper, MyTonWallet, Tonhub. Юзер подписывает proof — становится identity.

  • Ed25519 proof
  • Self-custody
  • Multi-wallet

E-commerce

Один аккаунт для входа на разных площадках экосистемы. Wishlists, история, рейтинг — переносятся.

  • Cross-store checkout
  • Quest система
  • Reviews привязаны к ID

SaaS

Замените самописную auth на стандартный OIDC. Меньше кода, меньше уязвимостей, выше конверсия.

  • OAuth 2.0 + PKCE
  • Готовые SDK
  • 5 минут до интеграции

Mobile apps

Telegram-вход без OAuth-редиректов — нативный flow в Telegram WebApp. Один тап.

  • Native deep-links
  • Biometric 2FA
  • Mini App SDK

Telegram Mini Apps

Если ваш сервис — Mini App в Telegram, Libermall ID работает нативно: HMAC initData валидируется автоматически.

  • initData валидация
  • Bot API интеграция
  • chat_id sync

Communities / Forums

Импорт пользователей через Libermall ID. Кросс-комьюнити репутация — отзывы Sites.Reviews видны в форуме.

  • Reputation API
  • Verified badge
  • Anti-bot защита
