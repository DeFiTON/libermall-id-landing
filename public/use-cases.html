Use cases

What products Libermall ID fits

5 use cases — from dApps to classic SaaS.

Web3 dApps

TON Connect login with Tonkeeper, MyTonWallet, Tonhub support. The user signs a proof — it becomes their identity. Powers dex.libermall.com.

  • Ed25519 proof
  • Self-custody
  • Multi-wallet

E-commerce

One account across every storefront in the ecosystem. Wishlists, history, ratings — they carry over.

  • Cross-store checkout
  • Quest system
  • Reviews tied to your ID

SaaS

Swap homegrown auth for standard OIDC. Less code, fewer vulnerabilities, higher conversion.

  • OAuth 2.0 + PKCE
  • Ready-made SDKs
  • 5 minutes to integrate

Mobile apps

Telegram sign-in without OAuth redirects — a native flow inside Telegram WebApp. One tap.

  • Native deep-links
  • Biometric 2FA
  • Mini App SDK

Telegram Mini Apps

If your service is a Telegram Mini App, Libermall ID is native: HMAC initData is validated automatically.

  • initData validation
  • Bot API integration
  • chat_id sync

Communities / Forums

Import users via Libermall ID. Cross-community reputation — Sites.Reviews ratings show up in the forum.

  • Reputation API
  • Verified badge
  • Anti-bot protection
