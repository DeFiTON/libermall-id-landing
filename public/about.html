About

Единая идентификация
для Web3-экосистемы

Libermall ID решает фрагментацию аккаунтов между продуктами экосистемы Libermall. Один профиль работает везде, история переносится, не нужно регистрироваться по сто раз.

Mission

Зачем мы это делаем

🎯 Миссия

Экосистема Libermall — это сеть взаимосвязанных продуктов: маркетплейс, биржа, виртуальные карты, каталог отзывов, AI-сервисы. Каждый работает лучше когда знает контекст пользователя — какие сайты вы оценили, какие монеты держите, какой рейтинг у вас как трейдера. До Libermall ID каждый сервис вёл свою отдельную базу аккаунтов — связи между ними рвались.

⚡ Решение

Libermall ID — один профиль, работающий везде. История переносится между продуктами. Не нужно регистрироваться по сто раз. Telegram-вход за один тап, TON Connect для криптокошельков, email magic-link для классики. Стандартный OpenID Connect — никакого vendor lock-in.

Tech stack

На чём построено

Casdoor core

Open-source identity server (BSD 3-Clause). Поддерживает OpenID Connect, OAuth 2.0, SAML 2.0, LDAP federation. 11k+ stars на GitHub.

PostgreSQL 16 + Redis 7

Профиль и сессии — Postgres, кэш и nonce — Redis. Daily backup, шифрование at-rest.

JWT RS256

4096-bit ключ для подписи токенов. JWKS rotation. Token lifetime 7 дней.

Architecture

Как это работает изнутри

Стандартный OAuth 2.0 / OpenID Connect flow. Кастомные адаптеры для Telegram Login Widget и TON Connect proof.

✓ Phase 2 · Done

● Phase 3 · In progress

Brand & UX

○ Phase 4 · Next

NFT usernames + $MALL

Минтинг @handle.lbm через смарт-контракт на TON. Token economy для рейтинга экосистемы.

○ Phase 5 · Future

Cross-product fabric

Wishlists в marketplace привязаны к ID, рейтинг трейдера на DEX, накопления на Card, отзывы на Sites.Reviews — единая активность по экосистеме. Quest-система и achievements за действия в разных продуктах.

○ Phase 6 · Vision

Libermall Passport

Криптографический паспорт по образцу Telegram Passport от Pavel Durov: верифицированные документы (KYC) и биометрия под cryptographic seal. Привязывается к Libermall ID. Использовать можно везде где нужна верификация — крипто-биржи, банки, фриланс-платформы — без передачи реальных данных оператору.

○ Phase 7 · Vision

Libermall Mail (на TON Storage)

Каждый Libermall ID получает @id.libermall.com email с end-to-end шифрованием по образцу Proton Mail. Метаданные и зашифрованные тела писем хранятся в распределённой сети TON Storage — никакого централизованного провайдера. Цель — превратить email из «+1 ещё одна почта» в реальный privacy-инструмент с self-custody.
