Как это работает

Знакомый поток OAuth 2.0 / OpenID Connect — никаких сюрпризов для разработчиков и пользователей.

1

Регистрация

На любом сайте экосистемы нажмите «Войти через Libermall ID». Выберите Telegram или TON-кошелёк.

2

Один клик

Подтвердите вход одним тапом в Telegram. Libermall ID создаст ваш профиль автоматически.

3

Везде в экосистеме

Этот же аккаунт работает на всех сайтах Libermall — без повторных регистраций.
