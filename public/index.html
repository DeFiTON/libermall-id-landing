Зачем нужен Libermall ID

Каждый продукт экосистемы Libermall больше не требует отдельной регистрации. Один аккаунт — всё под рукой.

Мгновенный вход

Telegram, TON Connect, email или social — выберите удобный способ. Без паролей, без писем подтверждения.

Безопасность

Industry-standard OpenID Connect, OAuth 2.0, SAML. Двухфакторная аутентификация, audit log, encryption.

Кросс-продуктовый профиль

TON-кошельки, отзывы, рейтинг трейдера, накопления на Card — единая история активности.

Как это работает

Знакомый поток OAuth 2.0 / OpenID Connect — никаких сюрпризов для разработчиков.

1

Регистрация

На любом сайте экосистемы нажмите «Войти через Libermall ID». Выберите Telegram или TON-кошелёк.

2

Один клик

Подтвердите вход одним тапом в Telegram. Libermall ID создаст ваш профиль автоматически.

3

Везде в экосистеме

Этот же аккаунт работает на всех сайтах Libermall — без повторных регистраций.
