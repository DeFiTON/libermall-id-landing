Sign-in methods

Три способа войти. Один аккаунт.

Любой способ ведёт к одному и тому же профилю. Привяжите все три — каждый станет резервным.

Telegram Login

Один тап в @LibermallIDbot — и вы внутри. Никаких форм. HMAC-подписи как в Telegram Login Widget.

~3 сек вход · 0 полей

TON Connect

Подпишите proof в любом TON-кошельке: Tonkeeper, MyTonWallet, Tonhub. Сам кошелёк становится вашим identity.

Ed25519 proof · Self-custody

Magic Link

Если нет Telegram или TON-кошелька — введите email, получите ссылку, кликните. Никаких паролей.

15 мин ссылка · 1-click
For developers

Стандартный OpenID Connect.
Нулевой vendor lock-in.

Подключите свой сервис к Libermall ID за 5 минут. Готовые SDK или сырой OIDC — на выбор.

  • OpenID Connect 1.0 — discovery, JWKS, userinfo
  • OAuth 2.0 Authorization Code + PKCE
  • SAML 2.0 для enterprise
  • JWT RS256, 4096-bit ключ
  • SDK: Laravel, Next.js, Node, Python
Документация GitHub
JavaScript Laravel Python curl 
// Авторизуйте пользователя за 5 строк
import { Issuer } from 'openid-client';

const issuer = await Issuer.discover(
  'https://id.libermall.com'
);

const client = new issuer.Client({
  client_id: process.env.LIBERMALL_CLIENT_ID,
  client_secret: process.env.LIBERMALL_SECRET,
  redirect_uris: ['https://yourapp.com/cb'],
});

// → Получите id_token с tg_id, name, email, wallets[]
