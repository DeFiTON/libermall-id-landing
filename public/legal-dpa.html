Data Processing Agreement (DPA)

Эффективен с 2026-05-24. Версия 1.0. Заключается с enterprise-клиентами в дополнение к Terms of Service.

1. Стороны

Контроллер: Клиент (компания, использующая Libermall ID для своих пользователей).
Процессор: Libermall (оператор Libermall ID).

2. Предмет обработки

  • Идентификаторы пользователей (Telegram ID, TON wallet, email)
  • Профильные данные (имя, аватар, language)
  • Сессии и audit log
  • OAuth consents

3. Цель обработки

Предоставление сервиса единой аутентификации согласно Terms of Service и инструкциям Контроллера.

4. Категории субъектов

Конечные пользователи Контроллера, использующие Libermall ID для входа в сервисы Контроллера.

5. Sub-processors

Libermall использует следующих sub-processors:

  • Fornex — VPS hosting (Frankfurt, DE)
  • Cloudflare — DNS, CDN, DDoS защита
  • Let's Encrypt — SSL сертификаты
  • GitHub — хранение исходного кода (не данных)

При добавлении новых sub-processors Контроллер уведомляется за 30 дней.

6. Меры безопасности

  • Encryption at rest (PostgreSQL data + backups)
  • Encryption in transit (TLS 1.3)
  • Access control (RBAC, 2FA для admins)
  • Daily backups с шифрованием
  • Audit log с 12-month retention
  • Incident response plan

7. Передача данных

Все данные хранятся в EU (Frankfurt, DE). Передача за пределы EU не осуществляется.

8. Права субъектов

По запросу Контроллера в течение 30 дней Libermall:

  • Экспортирует данные субъекта (GDPR Article 15)
  • Исправляет неточные данные (Article 16)
  • Удаляет данные (Article 17)
  • Передаёт данные в machine-readable формате (Article 20)

9. Утечки данных

В случае data breach Libermall уведомляет Контроллера в течение 24 часов и предоставляет:

  • Описание характера утечки
  • Категории и количество затронутых субъектов
  • Принятые и планируемые меры
  • Контакт DPO для дополнительных запросов

10. Аудит

Enterprise-клиенты имеют право на ежегодный аудит обработки данных, согласованный заранее. Альтернатива — независимые аудит-отчёты (SOC 2 Type 1 — Q4 2026, ISO 27001 — 2027).

11. Прекращение

При прекращении сотрудничества Libermall в течение 30 дней либо удаляет данные, либо возвращает их Контроллеру в зависимости от выбора Контроллера.

12. Контакт

DPO Libermall: dpo@libermall.com

Для подписи DPA: legal@libermall.com
