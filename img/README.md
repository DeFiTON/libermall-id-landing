# Image assets

Screenshots, diagrams and social previews used across the repo.

Note: the live brand SVGs (logo, favicon, OG cover) live in [`../public/assets/`](../public/assets/) so they're served directly by the static site. This folder is for documentation-side imagery only — README screenshots, architecture diagrams, conference slides, etc.

| File | Used by |
|---|---|
| `social-preview.png` | GitHub repository social preview (1280×640) |
| `arch-overview.svg` | `../docs/architecture.md` — high-level diagram |
| `screen-login.png` | `../README.md` — landing screenshot |

When you add an image, drop it here and reference with a relative path like `img/file.png` from the repo root.
